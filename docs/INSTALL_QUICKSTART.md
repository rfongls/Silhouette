## Prerequisites
- **Python**: 3.10+
- **Git**
- **Build tools** if needed on your OS (e.g., Visual C++ Build Tools on Windows)

## Install
- **Windows**
  ```bat
  py -m pip install -U pip
  py -m pip install -r requirements.txt
  ```
- **macOS/Linux**
  ```bash
  python3 -m pip install -U pip
  python3 -m pip install -r requirements.txt
  ```
